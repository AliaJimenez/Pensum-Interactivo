# Pensum Interactivo ITLA 🎓💻

¡Una herramienta web interactiva diseñada para que los estudiantes del Instituto Tecnológico de Las Américas (ITLA) puedan visualizar, planificar y hacer un seguimiento dinámico de sus asignaturas! 

El objetivo principal es resolver la dificultad de revisar los planes de estudio tradicionales, ofreciendo una experiencia visual y fluida para organizar el progreso académico cuatrimestre por cuatrimestre.

🚀 **Ver proyecto en vivo:** [Visitar Pensum Interactivo](https://pensumsinteractivositla.netlify.app/)

---

## ✨ Características
* **Visualización Dinámica:** Interfaz limpia y amigable que organiza las materias por cuatrimestre de forma clara.
* **Interactividad Enfocada:** Permite interactuar con el mapa curricular para una mejor planificación del cuatrimestre.
* **Diseño Responsivo:** Optimizado para que los estudiantes puedan consultar su progreso tanto desde la PC como desde el celular.
* **Orientado al Estudiante:** Pensado específicamente para la comunidad del ITLA, facilitando el control de la carrera.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Despliegue:** Netlify
